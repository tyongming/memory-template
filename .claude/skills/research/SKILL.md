# /research — External Research

## Trigger
Agent needs information not available in the codebase or memory-bank.

## Steps

1. Use web search to find relevant documentation, Stack Overflow answers, or library docs.
2. Summarize findings relevant to the current task.
3. Log to findings.md under "## External Research".
4. If the finding is reusable, log to `.learnings/LEARNINGS.md`.
