# Performance Log

> **Purpose:** Track agent performance across sessions for continuous improvement.
> **Update rule:** Write after each review cycle or significant agent interaction.

## Format

Each entry:

### {YYYY-MM-DD} — {agent-name} — {task-id}
**Verdict:** {APPROVE | APPROVE WITH COMMENTS | REQUEST CHANGES}
**Strengths:** {what went well}
**Issues:** {what needs improvement}
**Action items:** {specific changes for next session}
