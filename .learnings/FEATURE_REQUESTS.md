# Feature Requests & Missing Capabilities

> **Scope:** Things the workflow system can't do yet but should.
> **Write rule:** Add when you encounter a missing capability during a task.
> **Promotion rule:** If viable → create as new skill or agent config enhancement.

## Format

Each entry:

### {YYYY-MM-DD} — {short title}
**Task:** {TASK-ID}
**Situation:** {what the agent was trying to do}
**Gap:** {what capability is missing}
**Proposed solution:** {new skill / config change / tool addition}
**Priority:** {high — blocks workflow | medium — workaround exists | low — nice to have}
**Implemented:** {yes — date | no}
